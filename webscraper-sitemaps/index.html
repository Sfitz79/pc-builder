18 sitemaps generated for categories needing image re-scraping

How to use:

  1. Open Chrome with the WebScraper extension installed
  2. Click the WebScraper icon → Open WebScraper (or go to DevTools → WebScraper tab)
  3. Click "Import sitemap" → select one JSON file from webscraper-sitemaps/
  4. The sitemap will appear. Before scraping, add pagination: click "Edit" on the sitemap → "Add selector" → type: SelectorLink, id: next_page, selector: .pagination a[rel="next"] (or look for the > Next button), parent: _root, click: clickOnceOrUnclickable
  5. Click "Scrape" → let it run (may take a while for large categories like case/monitor/keyboard)
  6. When done, click "Export data as CSV"
  7. Save the CSV to C:\\Users\\simon\\Downloads\\ with a name starting with uk-pcpartpicker-com-
  8. Repeat for each category you want to rescrape
  9. After all categories are done, run: node merge-scraped-csvs.mjs
  10. Then run: node download-thumbnails.mjs
  11. Then: cmd /c npx vite build to verify the app builds

⚠ Pagination setup required

PCPartPicker uses hash-based pagination (#page=2, #page=3). After importing the sitemap, you must add the pagination selector manually:

Edit the sitemap → click "Add selector" → type: SelectorLink, id: next_page, parent: _root, click: clickOnceOrUnclickable, selector: the "Next" page button's CSS selector (typically .pagination a[rel="next"] or look for the > arrow link)

💡 Tip: Skip categories with good data

If a category already has enough products from the old scrape, you can skip it. Only the 19 categories above are missing images.
